# anhnbh.github.io

## Login Guard (Non-tech Protection)

This repo now uses a frontend login gate for all HTML pages.

- Login URL: `/login.html`
- Config file: `/assets/js/config.js`
- Default credentials:
  - Username: `admin`
  - Password: `123456`

To change credentials, edit:

`/assets/js/config.js`

## Important Note

This is only a client-side barrier to block non-technical users.
It is not strong security, because source code and content are still publicly accessible on GitHub Pages.
